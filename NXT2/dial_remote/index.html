Dial Remote Control
Building:
Program:

This simple remote control can be used to control another NXT creation by Bluetooth wireless remote control, if you have two NXTs, or by wired remote control to a two-motor creation without another NXT. 
 

You can select one of five functions by pressing the Enter or arrow buttons on the NXT, the touch sensor, or no button, then control the speed, power, or some other aspect of that function by rotating the dial forward or backward. The Button and Speed are indicated both graphically and textually on the NXT screen.

 

Bluetooth wireless remote control can control the Forklift and similar projects if you have two NXTs.
 		 Wired remote control can control the Mini Rover project with just one NXT.
